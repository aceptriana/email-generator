## Install

Ubuntu/Kali
```
sudo apt install git
sudo apt install python
pip install colorama
```
Arch/Manjaro
```
sudo pacman -S git
sudo pacman -S python
pip install colorama
```
Windows
[Download Python ](https://www.python.org/downloads/)

## Run
```
git clone https://github.com/aceptriana/email-generator
cd email-generator
python main.py

```
